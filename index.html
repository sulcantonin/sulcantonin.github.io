01 / About

Profile

I am a researcher at Lawrence Berkeley National Laboratory, where I design intelligent algorithms and autonomous control systems for large-scale scientific infrastructure. My work bridges large language models, agentic systems, and high-energy physics — enabling self-optimizing operation in safety-critical environments where being wrong has real-world consequences.

Previously a Senior Data Scientist at the Helmholtz Association (DESY / European XFEL), I pioneered the use of domain-specific language models in scientific operations — from logbook understanding to anomaly detection on superconducting RF cavities.

I hold a Ph.D. in Computer Vision from the University of Konstanz, and I bring the mathematical rigor of high-dimensional signal processing to the multi-modal machine states of modern accelerators — turning noisy, heterogeneous telemetry into actionable, interpretable insight.

Berkeley Lab Ph.D. — U. Konstanz ex-DESY · XFEL Helmholtz Association

05 / Software

Open source

Reference implementations of the methods above — production-quality, pip-installable, MIT-licensed.

event2vector

PyPI · GitHub

A scikit-learn–style Python library providing a geometric approach to learning composable, highly interpretable representations of discrete event sequences.

PyPI ↗ GitHub ↗

torchmodal

PyPI · 2 repos

A framework introducing Differentiable Modal Logic for PyTorch — enabling the training of Modal Logical Neural Networks (MLNNs).

PyPI ↗ torchmodal ↗ MLNN ↗

FastLSQ

PyPI · paper

A lightweight framework for solving PDEs in one shot via Fourier features with exact analytical derivatives — bypassing autodiff and iterative training loops for linear and nonlinear problems.

PyPI ↗ GitHub ↗ Paper ↗

PACuna

GitHub · NeurIPS '23

Automated data collection and fine-tuning pipelines designed to adapt large language models, securely, to the complex domains of particle accelerators.
